FIX CSRF TOKEN ERROR
================================================================================

If you're getting CSRF token error on login:

1. Stop server (Ctrl + C)
2. Clear browser cache:
   - Chrome: Ctrl + Shift + Delete
   - Select "Cookies" and "Cache"
   - Click "Clear data"
3. Close ALL browser tabs
4. Start server: python manage.py runserver
5. Open NEW browser window
6. Go to: http://127.0.0.1:8000/accounts/login/
7. Login

OR just use incognito/private window!
