HOW IT WORKS
================================================================================

STEP 1: Admin Logs In
---------------------
- URL: http://127.0.0.1:8000/accounts/login/
- Login as admin
- Auto-redirected to: http://127.0.0.1:8000/admin/dashboard/

STEP 2: View Dashboard
----------------------
- See 4 statistics cards at top
- View all blood requests below
- See all donor responses for each request
- Filter requests by urgency
- Scroll down to see all registered donors

STEP 3: Create Blood Request
----------------------------
- Click "Create Blood Request" button
- Modal popup opens (beautiful animation)
- Fill form:
  * Blood Group
  * Units Needed
  * Urgency Level
  * Notes
- Click "Create Request"
- Modal closes
- Page refreshes with new request
- See success message at top

STEP 4: View Responses
----------------------
- Each request card shows:
  * Request details (blood group, units, urgency)
  * Number of notifications sent
  * All donor responses
  * Donor avatars
  * Contact information
  * Accept/Reject status

STEP 5: Filter & Search
-----------------------
- Filter requests by urgency (top buttons)
- Filter donors by blood group (bottom section)
- Click to toggle filters
